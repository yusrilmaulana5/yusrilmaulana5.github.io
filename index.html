MATERI KE 1

KOMPUTER DAN KOMPONEN

CPU (Central Processing Unit) adalah inti komputer yang mengolah instruksi dan data. Komponen utama CPU meliputi:

  1. Control unit: Mengendalikan operasi CPU dan komputer.
  2. ALU (Arithmetic Logic Unit): Melakukan pemrosesan data dan operasi matematika.
  3. Registers: Menyimpan data sementara dalam CPU.
  4. CPU interkoneksi: Mekanisme komunikasi antara unit kontrol, ALU, dan registers.

Single Core CPU dan Multi Core CPU memiliki komponen inti yang sama dengan CPU tunggal, tetapi Multi Core CPU memiliki beberapa inti pemrosesan yang bekerja bersama untuk meningkatkan kinerja.

ALU (Arithmetic Logic Unit) adalah bagian CPU yang melakukan operasi aritmatika (penjumlahan, pengurangan, dll.) dan operasi logika (AND, OR, dll.) dalam komputer.

  • Peran ALU: Melakukan operasi matematika, logika, perbandingan, bitwise, dan menangani overflow.

Registers adalah tempat penyimpanan data dalam CPU. Beberapa jenis register termasuk:

  1. MAR (Memory Address Register): Menyimpan alamat memori.
  2. MDR (Memory Data Register): Menyimpan data yang diambil dari atau akan ditulis ke memori.
  3. PC (Program Counter): Menunjukkan alamat instruksi berikutnya yang akan dieksekusi.
  4. Accumulator: Menyimpan hasil sementara operasi.
  5. Index Register: Digunakan untuk mengubah alamat operan selama eksekusi program.
  6. Instruction Register: Menyimpan instruksi yang dieksekusi atau yang akan didekodekan.

Masing-masing register memiliki peran khusus dalam operasi CPU dan pemrosesan data.


MATERI KE 2

Sejarah Komputer Generasi
  • Generasi I (1940-1956): Vacuum Tube, bahasa mesin (0 dan 1), besar.
  • Generasi II (1956-1963): Transistor, bahasa assembly, lebih kecil.
  • Generasi III (1964-1971): IC, ukuran lebih kecil.
  • Generasi IV (1971-sekarang): Mikroprosesor, layar monitor, beragam ukuran.
  • Generasi Selanjutnya: Cloud computing dan AI.
Gerbang Logika

Fungsi: Membentuk sirkuit digital dengan logika dasar.

Jenis Gerbang:

  • AND: Output 1 jika semua input 1.
  • OR: Output 1 jika ada setidaknya satu input 1.
  • NOT: Membalik keadaan input.

Transistor dan Semikonduktor

Transistor: Pengganti Vacuum Tube dalam Generasi II.

Semikonduktor: Digunakan dalam IC pada Generasi III.

Generasi IV Komputer

Menggunakan Mikroprosesor sebagai komponen digital.

Layar monitor dengan GUI. Beragam ukuran.

Generasi Selanjutnya

Terhubung dengan cloud computing dan AI.


MATERI KE 3

Apa Itu Memory?

Memory adalah sistem penyimpanan informasi langsung dalam komputer dan perangkat elektronik.

Fungsi Memory

Memiliki fungsi untuk menyimpan data yang dapat langsung digunakan dalam komputer.

Memory Hierarchy

Struktur berjenjang yang membedakan tingkatan memori berdasarkan kecepatan dan kapasitasnya.

Jenis Memory
  • Memory Read Only (ROM)
  • Memory Read/Write
  • Static Memory
  • Dynamic Memory
  • Volatile dan Non-volatile
Perkembangan Memory
  • RAM (1968)
  • DRAM (1970)
  • FP RAM (1987)
  • EDO RAM (1995)
Cara Kerja Memory

Memory menyimpan dan mengakses data berdasarkan alamat tertentu dalam unit yang disebut byte.

Contoh Optimalisasi Memory pada Bahasa Pemrograman

Strategi optimalisasi memori pada bahasa pemrograman termasuk penggunaan variabel lokal, alokasi memori yang efisien, dan penghapusan memori yang tidak terpakai.


MATERI KE 4

Data pada Komputer

Data adalah fakta, angka, atau informasi yang digunakan untuk pengambilan keputusan, analisis, atau tujuan lainnya. Bisa berupa teks, angka, gambar, audio, atau video.

Data Numerik

Data Numerik terbagi menjadi:

  • Data Numerik: Angka, seperti sistem Desimal dan Biner.
  • Data Non-Numerik: Teks, gambar, dan sebagainya.
Sistem Desimal

Sistem bilangan sehari-hari menggunakan angka 0-9. Misal: 83 = (8x10) + 3.

Sistem Biner

Sistem bilangan dengan angka 0 dan 1. Contoh: 1 = 0001, 8 = 1000.

Integers

Bilangan bulat, baik positif, negatif, atau nol. Dalam pemrograman, dinotasikan sebagai int.

Bilangan Real

Bilangan dengan desimal atau pecahan, seperti 3.14 atau -0.05, dinotasikan sebagai float/double.

Data Non-Numerik

Representasi teks melibatkan character encoding, teks mentah, markup languages, format dokumen, atau file teks.

Representasi gambar digital terdiri dari piksel dengan format yang dapat berkompresi seperti jpg, gif, png, dll.


MATERI KE 5

Bahasa Pemrograman

Bahasa pemrograman adalah alat yang digunakan untuk menulis kode yang dapat diinterpretasikan oleh komputer, memungkinkan komunikasi antara manusia dan mesin.

Fungsi Bahasa Pemrograman
  • Memudahkan komunikasi antara komputer dan pengguna.
  • Membuat aplikasi yang kompleks.
  • Membuat kode yang dapat digunakan kembali.
  • Mempercepat pengembangan perangkat lunak.
  • Mengizinkan komputer untuk melakukan berbagai tugas.
Bahasa Mesin/Bahasa Assembly

Bahasa paling rendah dalam tingkat abstraksi, menggunakan instruksi dalam kode biner yang langsung dieksekusi oleh komputer.

Bahasa Tingkat Rendah

Memberikan kontrol langsung atas perangkat keras dengan instruksi lebih dekat dengan bahasa mesin, namun lebih mudah dipahami oleh manusia. Contohnya: C dan C++.

Bahasa Tingkat Menengah

Menawarkan abstraksi lebih tinggi daripada bahasa tingkat rendah, digunakan untuk mengembangkan perangkat lunak sistem. Contohnya: Ada, Rust.

Bahasa Tingkat Tinggi

Jauh dari bahasa mesin, mudah dipahami manusia, dan digunakan untuk aplikasi web, mobile, dan data science. Contohnya: Python, Ruby, JavaScript.


MATERI KE 6

Translator

Dalam konteks bahasa pemrograman, "translator" mengubah kode sumber dari satu bahasa pemrograman ke lainnya. Ada dua jenis utama translator: Compiler dan Interpreter.

Compiler

Menerjemahkan seluruh kode sumber ke dalam bahasa mesin atau intermediate code. Melalui fase analisis (lexical, syntax, semantic analysis, intermediate code generation) dan fase sintesis (code optimization, code generation).

Contoh bahasa yang menggunakan compiler: C, C++, dan Ada.

Interpreter

Membaca dan mengeksekusi kode sumber baris demi baris saat program berjalan, tanpa menerjemahkan seluruhnya terlebih dahulu.

Contoh bahasa yang menggunakan interpreter: Python, JavaScript, dan Ruby.

Compiler vs Interpreter
Compiler Interpreter
Siap dijalankan Tidak lintas platform
Lebih cepat Fleksibel
Source tidak terlihat Source terlihat
Linker

Menggabungkan berbagai library external dengan object file untuk membuat program yang dapat dieksekusi.

Contoh, dalam bahasa C, perintah seperti printf() membutuhkan file external seperti stdio.h. Linker menggabungkan file-file tersebut.
